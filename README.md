# Python Projects for CV

A collection of Python projects demonstrating data processing, automation, and visualization skills.

## Projects

### 1. CSV Data Cleaner & Report Generator
Cleans messy CSV files and generates Excel + PDF summary reports.
```bash
cd 1_csv_cleaner
python clean_csv.py --input your_data.csv --output cleaned_output
```

### 2. File & Folder Organizer Bot
Monitors a folder and auto-sorts files by type with full action logging.
```bash
cd 2_file_organizer
python file_organizer.py --watch "C:/Users/YourName/Downloads" --existing
```

### 3. Interactive Engineering Data Dashboard
Streamlit dashboard for visualizing datasets with Plotly charts and KPI cards.
```bash
cd 3_dashboard
streamlit run dashboard_app.py
```

## Setup
```bash
pip install -r requirements.txt
pip install statsmodels
```
